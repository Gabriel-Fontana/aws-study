# Módulo 6 & 7 – Storage & Databases 💾

### 📦 Armazenamento de Objetos (S3)
* **Simple Storage Service:** 11 noves de durabilidade (99.999999999%).
* **Classes:** Standard (dia a dia), Intelligent-Tiering (IA decide o custo), Glacier (arquivamento).

### 📊 Bancos de Dados Gerenciados
* **RDS:** Relacional (SQL). Backup, multi-AZ e patch automáticos.
* **DynamoDB:** NoSQL de performance absurda. Milissegundos de latência.
* **Aurora:** O banco "cloud native" da AWS. 5x mais rápido que o MySQL padrão.
