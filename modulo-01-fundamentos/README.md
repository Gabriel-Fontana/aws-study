# Módulo 1 – Introduction to the Cloud ☁️

### 📌 Conceitos-Chave
* **Nuvem Pública:** Infraestrutura da AWS compartilhada, escalável e paga pelo uso. Ideal para agilidade.
* **Nuvem Privada:** Exclusiva da empresa. Maior controle, mas exige investimento pesado (CapEx).
* **Nuvem Híbrida:** O mix estratégico. Mantém dados sensíveis localmente e escala o restante na AWS.

### 💰 Benefícios Financeiros
* **CapEx → OpEx:** Paramos de gastar com hardware físico (Capital Expenditure) e passamos a ter gastos operacionais variáveis.
* **Escalabilidade:** Só pagamos pelo que usamos, com acesso global instantâneo.

> **Resumo Estratégico:** A nuvem não é apenas tecnologia, é uma mudança de modelo de negócio para ganhar velocidade e reduzir desperdício.
